# malachite

Import Last.fm and Spotify listening history to AT Protocol as `fm.teal.alpha.feed.play` records. Web interface at [malachite.croft.click](https://malachite.croft.click).

Full documentation at **[docs.ewancroft.uk](https://docs.ewancroft.uk/projects/malachite)**.
